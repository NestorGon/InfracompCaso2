Para correr el programa debe asegurarse de que tenga la JRE System Library con jdk-15

El metodo main se encuentra en la clase Monitor. A la hora de ejecutar el proyecto, debe ingresar la ruta donde se encuentra el archivo de datos, un ejemplo de ruta sería el siguiente:

./data/2021_20_referencias_caso_2/referencias8_128_75.txt

El número que se imprime posteriormente es el número de fallas de página que ocurrieron.
